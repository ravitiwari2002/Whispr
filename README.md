*Whispr - Hear the Whispers* 

Whispr is an application designed to foster anonymous connections, allowing users to engage in conversations with random individuals in secure chatrooms. This platform provides a unique opportunity for spontaneous interactions and the exchange of diverse perspectives, all while maintaining the userâ€™s anonymity.

## Features

The following functionalities are implemented:

-  Account creation and login functionality for users.
-  Capability for users to create chatrooms.
-  Search functionality for users to find and join chatrooms.
-  Feature for users to view the list of active members within a chatroom.
-  Option for users to exit chatrooms at their discretion.
